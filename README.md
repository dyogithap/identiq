# IdentIQ

IdentIQ is an AI-powered Intelligent Digital Vault designed to securely store, analyze, and manage sensitive personal documents like passports, ID cards, and licenses.

## Features
- Automated PII extraction using AI
- Document expiry tracking
- Smart reminders for expiring documents
- Fraud detection using confidence scores

## Tech Stack
- Flutter (Frontend)
- Dart (Strictly typed, OOP-based)
- Google Cloud Document AI
